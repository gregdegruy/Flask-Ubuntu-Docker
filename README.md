# Let's Flask it Up!

### Docker style

#### To Build: 
`$ docker build -t flaskweb:latest . `

#### To Start: (interactive)
`$ docker run -p 5000:5000 -i --rm flaskweb:latest` 

#### To See the running instance:
`$ docker ps`


#### To Kill:

`$ docker kill [thirsty_mayer or container id - found by running docker ps]`
